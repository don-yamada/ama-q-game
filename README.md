# ama-q-game

## aws_service_sorter_game_v2

### 遊び方

ゲームモードと難易度を選択してスタート。
仕分けはカーソルキー（←↑→）で行います。

間違えずに正解を続けることでコンボが発生し、10 コンボごとにボーナスタイムをゲットできます。
仕分けを間違えると −3 秒のペナルティです。

ゲームクリア後に称号があり、全 10 種類のはずです。

### 起動

`python aws_service_sorter_game_v2.py`

### 環境

`pip install pygame`
